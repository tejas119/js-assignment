Data Type

What does Data Type mean?

Data Types basically specify what kind of data can be stored and manipulated within a program.

Their are 6 primitive data type and 3 reference data type

What are primitive and reference data type

primitive Data type

    1.Number
    2.String
    3.Bigint
    4.boolean
    5.undefined
    6.null

reference Data type

    1.Array
    2.object
    3.Function

We will take one by one type

    1.Number

    ECMAScript has two built-in numeric types: Number and BigInt

    It contain number in it eg:123456

    2.String

    String type is used to represent textual data.

    It is a set of "element" of 16-bit unsigned integer values.

    However, it is still possible to create another string based on an operation on the original string.

    For Example:

    A concatenation of string using the concatenation operator (+) or string.concat().


    3.Bigint

    The Bigint type is a numeric primitive

    you can safely store and operate on large integers even beyond the safe integer limit for Numbers


    4.Boolean

    Boolean represents a logical entity and can have two values: true and false


    5.undefined

    A variable that has not been assigned a value has the value undefined


    6.null

    The Null type has exactly one value:null
